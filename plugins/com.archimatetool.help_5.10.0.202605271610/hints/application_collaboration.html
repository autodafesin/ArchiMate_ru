Кооперация приложений представляет собой агрегат из двух или более элементов внутренней активной структуры приложения, которые совместно работают для выполнения согласованного поведения приложения.

Категория:

Приложение. Активная структура.

Примеры:

Два компонента приложения осуществляют кооперацию при администрировании транзакций: компонент бухгалтерского учета и компонент выставления счетов. Эта кооперация выполняет задачу «Администрировать транзакции».
